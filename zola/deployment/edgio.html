Edgio

如果您没有 Edgio 帐户，可以在此处注册。

🔗手动部署

对于命令行手动部署，请执行以下步骤：

  1. 安装 Edgio CLI：
npm i -g @edgio/cli
  1. 使用以下内容在项目的根目录下创建一个 package.json：
npm init -y
  1. 使用以下命令初始化您的项目：
edgio init
  1. 将项目根目录中的 routes.js 更新为以下内容：
// This file was added by edgio init.
// You should commit this file to source control.

import { Router } from '@edgio/core/router'

export default new Router().static('public')
  1. 构建您的 zola 应用程序：
zola build
  1. 部署！
edgio deploy
