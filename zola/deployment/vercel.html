Vercel

Vercel（以前称为 Zeit）类似于 Netlify，使您的站点部署变得非常容易。这些站点由 Vercel 托管，并在我们将提交推送到我们选择的生产分支（例如，master）时自动部署。

如果您没有 Vercel 帐户，可以在此处注册。

🔗自动部署

注册后，您可以从 Git 提供商（Github、GitLab 或 Bitbucket）导入您的网站。当您导入存储库时，Vercel 将尝试找出您的网站使用的框架。

如果它不默认为 Zola：

  • 将框架预设设置为Zola。

默认情况下，Vercel 选择输出目录为public. 如果您使用不同的目录，请在“构建和输出设置”下拉列表下指定输出目录。您可以通过他们的文档了解有关如何设置自定义域以及如何充分利用 Vercel 的更多信息 。

单击蓝色的“部署”按钮后，比赛就开始了！
